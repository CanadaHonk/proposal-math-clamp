Introduction

`Math.clamp()` constrains a value between an upper and lower bound.

Math.clamp ( _number_, _min_, _max_ )

The `clamp` function returns a number that is the result of constraining _number_ between the lower bound defined by _min_ and the upper bound defined by _max_.

1. Let _n_ be ? ToNumber(_number_). 1. Let _mn_ be ? ToNumber(_min_). 1. Let _mx_ be ? ToNumber(_max_). 1. If _mn_ > _mx_, throw a *RangeError* exception. 1. If _n_ < _mn_, return _mn_. 1. If _n_ > _mx_, return _mx_. 1. Return _n_.
